# แบบสอบถามโบนัสทายปัญหา — FarmSim EDU

คำถามแบบ **ใช่ / ไม่ใช่** (A / B) สำหรับโบนัสทายปัญหาระหว่างเกม

## คะแนนเหรียญ

| ผล | คะแนน |
|----|--------|
| ตอบถูก **คนแรก** | **+10** เหรียญ |
| ตอบถูกคนถัดไป | **+2** เหรียญ |
| ตอบผิด | **-10** เหรียญ |

> ตอบเร็วได้คะแนนมากกว่า · หมดเวลา 25 วินาทีหรือทุกคนตอบครบแล้วปิดอัตโนมัติ

## ตัวเลือก

- **A. ใช่**
- **B. ไม่ใช่**
